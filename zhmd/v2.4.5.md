## 概述

文档版本：提供完整中文 README，并在英文与中文 README 及发行说明页面上提供双语语言切换。节点代码与运行时行为无变更。

---

## 新增内容

### 中文文档（`zhmd/`）

- **`zhmd/README.md`**：仓库 README 的中文译文（安装、功能、节点截图、更新日志与链接）。
- **`zhmd/v2.4.4.md`**：v2.4.4 中文发行说明（与英文 v2.4.4 发行页配对）。
- 本系列后续版本的发行说明放在 `zhmd/` 下（例如本文件 `zhmd/v2.4.5.md`）。

### 双语语言切换

- 根目录 **`README.md`**：EN ↔ 中文 切换（HTML 表格 + PNG 图标），链接至 `zhmd/README.md`。
- **`zhmd/README.md`**：中文 ↔ EN 切换，链回英文 README。
- **发行说明**（GitHub Releases 与仓库内副本）：英文（`RELEASE_NOTES/v*.md`）与中文（`zhmd/v*.md`）页面均采用相同切换器。

### GitHub 显示修复

- **`zhmd/README.md`**：节点截图路径使用 `../images/`，在 GitHub 上正确指向仓库根目录的 `images/`（修复原先 `images/` 误解析为 `zhmd/images/` 导致图片无法显示的问题）。

---

## 文档目录结构

| 英文 | 中文 |

两份 README 的更新日志在 **Changelog** / **更新日志** 小节保持对应。

## 升级说明

- 若仅使用节点，**无需安装或更新代码**；若需要中文文档，拉取或刷新仓库即可。
- 可从主 README 上的切换器打开 [中文 README](https://github.com/ussoewwin/ComfyUI-QwenImageLoraLoader/blob/main/zhmd/README.md)。
